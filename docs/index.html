How it works

Three steps. Every AI knows you.

1

Scan your machine

aura detects your languages, frameworks, tools, projects, and editor automatically. Zero YAML required.

aura scan # or the full flow: aura quickstart
2

Answer 5 questions

Your role, communication style, current priorities, and rules. 2 minutes to capture what scanning can't.

aura onboard # or add facts directly: aura add dev tone "Direct"
3

Serve via MCP

Start the MCP server. Claude, ChatGPT, Cursor, and Gemini read your context automatically.

aura setup # config tools aura serve # start MCP

The real power:
built-in MCP

aura includes a full MCP server. Start it once and every compatible tool — Claude, ChatGPT, Cursor, Gemini — reads your context automatically.

No copy-paste. No re-explaining. Your context packs are live resources that AI tools query in real time.

Works with any tool that supports the Model Context Protocol.
